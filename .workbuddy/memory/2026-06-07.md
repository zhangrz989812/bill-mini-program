## 技术笔记

### WXS 使用规范
- WXML 不支持直接调用 JS 方法（如 `.toFixed(2)`）
- 需要使用 WXS 模块进行数据格式化
- WXS 语法类似 ES5，不支持 ES6+ 特性

### 用户信息获取
- 旧版 `wx.getUserInfo` 已废弃
- 新版使用 `open-type="chooseAvatar"` 获取头像
- 使用 `type="nickname"` 的 input 获取昵称
- 需要将用户信息保存到本地缓存

### TDesign 集成
- 需要先 `npm install tdesign-miniprogram`
- 在微信开发者工具中「构建 npm」
- 确保 `app.json` 中没有 `"style": "v2"`
- 最低基础库版本要求：`^2.12.0`
